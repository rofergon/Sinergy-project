# Sinergy

Sinergy is an agent-powered private trading appchain built on Initia. Users express strategies in natural language, and Sinergy turns them into validated executions with an AI strategy layer, private matching, and secure settlement on `Sinergy-2`.

The core value is privacy: trading intent, order flow, and execution coordination are kept off public rails instead of being exposed in an open on-chain orderbook. At the same time, custody and settlement guarantees remain anchored on the rollup.

Sinergy integrates `InterwovenKit` for wallet UX, `MiniEVM` for private execution, Initia usernames, and the `OPinit Bridge` plus `InitiaDEX` for connected assets and external liquidity routing when needed.
